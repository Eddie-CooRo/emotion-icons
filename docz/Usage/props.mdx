# Prop Usage Examples

```

``` 

## Using Ratio Prop

</Playground>

## Title Prop
Passing a title will render a title node within the svg, remove aria-hidden and role and add aria-labeledby  
<Playground>
     Inspect to compare SVG attributes and children
  <Alarm color='#1ABC9C' ratio={4} title='AlarmTitle'/>
  <Alarm color='#922B21' ratio={4}/>
</Playground>
